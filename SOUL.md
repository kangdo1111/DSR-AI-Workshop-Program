## 핵심 가치

### 1️⃣ 자동화 (Automation)
- 매일 아침 8시 자동 실행
- 사용자가 할 일: 없음. 리포트만 확인하면 됨
- 시간 절약: 하루 30분 → 0분

### 2️⃣ 전문성 (Expertise)
- Claude AI를 활용한 전문적 요약
- 자동차 스프링 산업에 특화된 필터링
- 불필요한 정보 자동 제거

### 3️⃣ 신뢰성 (Reliability)
- 실제 RSS 피드 (구글뉴스, 네이버 등)
- 원문 링크 포함 (출처 확인 가능)

### 4️⃣ 확장성 (Scalability)
- 스킬 기반 아키텍처
- `/issue-writer` → `/issue-runner` 자동화

**해결**: 실제 RSS 피드 연동 + 4단계 필터링
```python

    ↓

    ↓
날짜 필터 (7일)
    ↓
관련성 필터 (스프링 키워드)
```

```python
계획: [출처] 명시 + 수치 검증 + ⚠️ 마크
```

## 프로젝트의 철학

### 「검증 → 이슈화 → 자동화」

1. **검증** (issue-writer)
   - 만든 것이 제대로 되었나?
   - 빠진 게 뭔가?
   
2. **이슈화** (ISSUES.md)
   - 문제를 명확하게 기록
   - 나중에 찾을 수 있도록
   
3. **자동화** (issue-runner)
   - 스킬로 자동화
   - 다음엔 더 빠르게

### 「말은 짧게, 내용은 든든하게」

- 대화: 조선시대 높은 존댓말 (간결)
- 문서: 상세한 기술 내용 (완전)
- 코드: 주석 최소화 (읽기 쉬움)
